# `func` в Go

---

## Краткий ответ на собеседовании

В Go функции объявляются с помощью ключевого слова `func`.  
Функции могут принимать параметры, возвращать одно или несколько значений
и являются основным способом организации логики в программе.

---

## Объявление функции

```go
func greet() {
    fmt.Println("Hello")
}
```
- функция объявляется через func
- имя функции пишется в camelCase
- тело функции всегда в фигурных скобках

## Функция с параметрами
```go
func greet(name string) {
    fmt.Println("Hello,", name)
}
```
-параметры указываются в скобках
-тип параметра указывается после имени

## Функция с возвращаемым значением
```go
func sum(a int, b int) int {
    return a + b
}
```
- тип возвращаемого значения указывается после параметров
-  return завершает выполнение функции

## Сокращённая запись параметров одного типа
```go
func sum(a, b int) int {
    return a + b
}
```

## Возврат нескольких значений
func divide(a, b int) (int, bool) {
    if b == 0 {
        return 0, false
    }
    return a / b, true
}
- функции в Go могут возвращать несколько значений
- часто используется для возврата результата и ошибки

## Именованные возвращаемые значения
```go
func calc(a, b int) (result int) {
    result = a + b
    return
}
```
- возвращаемые значения объявляются заранее
- return можно писать без указания переменных

## Ранний return
```go
func check(x int) {
    if x < 0 {
        return
    }
    fmt.Println("x положительный")
}
```
- Используется для упрощения логики и уменьшения вложенности.

## Функции как значения
```
func add(a, b int) int {
    return a + b
}
var f func(int, int) int = add
```
- функции можно присваивать переменным
- функции можно передавать как аргументы


## Анонимные функции
```
func() {
    fmt.Println("anonymous")
}()
```
- функция без имени
- может быть вызвана сразу

## Когда использовать функции

- для разделения логики  
- для переиспользования кода  
- для улучшения читаемости  

---

## Короткая формула для запоминания

- `func` — объявление функции  
- функция = параметры + возвращаемые значения  
- функции — основной строительный блок Go
