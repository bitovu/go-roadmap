# `var` и `const` в Go

---

## Краткий ответ на собеседовании

В Go для объявления переменных чаще всего используется короткий синтаксис `:=`.  
`var` применяется реже — когда значение появляется позже, нужен конкретный тип
или объявление происходит вне функции.  
`const` используется для неизменяемых значений, известных на этапе компиляции.

---

## Как объявляют переменные в реальном Go-коде

### Короткое объявление (`:=`) — основной способ

```go
x := 10
name := "Go"  
ok := true
```
* используется по умолчанию
* работает только внутри функций
* тип определяется компилятором и фиксируется

---

## Когда действительно нужен `var`

### Значение появляется позже

```go
var result int
result = calculate()
// Переменная объявляется заранее,
а значение присваивается позже.
```
### Нужен конкретный тип

```go
var x int32 = 10
```

###Объявление нескольких переменных
```
var a, b int = 1, 2

var (
    count  int
    title  string
    active bool
)
```
