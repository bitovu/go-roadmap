# Примитивные типы данных в Go

## Короткий ответ (на собесе)

> В Go примитивные типы данных делятся на **числовые, логический и строковый**.

- Числовые: целочисленные, вещественные и комплексные  
- Логический тип — `bool`  
- Строковый тип — `string` (иммутабельный, UTF-8)  
- `rune` — это `int32`, представляющий Unicode-кодпоинт  

---

## Развёрнутый ответ

### Числовые типы

- **Целочисленные**: `int`, `uint`, `int8–int64`, `uint8–uint64`
- **Вещественные**: `float32`, `float64`
- **Комплексные**: `complex64`, `complex128`

### Логический тип
`bool` хранит `true` и `false`.

### Строки
`string` — иммутабельный тип, хранится в UTF-8.  
`len(string)` возвращает **байты**, а не символы.

### Rune
`rune` — это числовой тип (`int32`), представляющий Unicode-символ.

---

## Пример

```go
package main

import "fmt"

func main() {
    s := "привет"

    fmt.Println(len(s))         // 12
    fmt.Println(len([]rune(s))) // 6

    var r rune = 'Ж'
    fmt.Println(r) // 1046
}
